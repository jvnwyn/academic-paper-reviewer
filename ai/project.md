# Project Identity & Purpose

## Identity
- **Name:** Academic Paper Reviewer
- **Type:** RAG-based Academic Paper Q&A Web Application
- **Stack:** Python 3.14+, Flask 3.1, PyMuPDF, sentence-transformers (`BAAI/bge-small-en-v1.5`), ChromaDB, Google Gemini (`gemini-3.6-flash`)

## Goal
To allow users to upload PDF research papers and ask questions, receiving answers strictly grounded in the document text with exact page citations (`[Page N]`), preventing hallucinations.
